# Queue file — one TikTok video ID per line
# Lines starting with # are comments and are ignored
# Blank lines are also ignored
#
# How to use:
#   npx tsx src/index.ts queue-tiktok --file scripts/tiktok/queues/example.txt
#
# How to find video IDs:
#   - In DBeaver: browse tiktok.tiktok_videos, copy the video_id column
#   - From a TikTok URL: https://www.tiktok.com/@handle/video/7459779252146228497
#                        the ID is the number at the end
#
# Create your own files in this folder, e.g.:
#   scripts/tiktok/queues/business-ideas.txt
#   scripts/tiktok/queues/cooking-content.txt
#   scripts/tiktok/queues/interesting-oct-2026.txt
#
# Example IDs (replace with real ones from your library):
# 7459779252146228497
# 7374814047495081248
# 7450201371153861934
